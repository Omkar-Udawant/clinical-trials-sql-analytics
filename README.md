# Clinical Trials SQL Analytics

This project analyzes clinical trial data using SQL and PostgreSQL.

The goal is to transform raw clinical trial data into a normalized database
and generate insights about medical research trends.

## Tools Used

- PostgreSQL
- SQL
- Power BI (coming next)

## Database Schema

The database includes the following tables:

- organizations
- trials
- conditions
- interventions
- trial_conditions
- trial_interventions

## Analysis Performed

- Top organizations conducting trials
- Most researched diseases
- Clinical trial phase distribution
- Study type analysis
- Most common interventions

## Dataset

Source: ClinicalTrials.gov dataset (Kaggle)

Development dataset: 1000 rows  
Full dataset: ~496,000 rows
